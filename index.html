üîß Troubleshooting Guide

Common issues and their solutions

üêç Python Version Issues
Problem: "Python command not found" or version too old
Solution: Install Python 3.8+ from python.org or use pyenv for version management
# Check Python version python --version python3 --version # If using python3 python3 -m venv fuel_pricing_ml_app_venv
üì¶ Package Installation Failures
Problem: pip install errors or missing dependencies
Solution: Upgrade pip and use virtual environment
# Upgrade pip python -m pip install --upgrade pip # Install in virtual environment source fuel_pricing_ml_app_venv/bin/activate pip install -r requirements.txt
üåê Port Already in Use
Problem: "Port 8001 or 8501 already in use"
Solution: Use different ports or stop conflicting processes
# Use different port for FastAPI uvicorn app.main:app --host 0.0.0.0 --port 8002 # Use different port for Streamlit streamlit run streamlit_app.py --server.port 8502
üß† Model Training Errors
Problem: Model training fails or data file not found
Solution: Ensure data files exist and check file paths
# Check if data file exists ls backend/data/raw/oil_retail_history.csv # Verify working directory pwd
üîó API Connection Issues
Problem: Streamlit cannot connect to FastAPI backend
Solution: Ensure both services are running and check URLs
# Check if FastAPI is running curl http://localhost:8001/health # Check Streamlit configuration in streamlit_app.py # Ensure API_BASE_URL is correct
